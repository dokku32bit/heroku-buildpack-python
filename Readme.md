Dokku x86 buildpack: Python
========================

This is buildpack is based on the original Heroku Python buildpack ([https://github.com/heroku/heroku-buildpack-python]). The python runtimes have been replaced with x86 ones


Provided Runtimes
-----------------

- python-2.7.6
- python-2.7.7
- python-2.7.8
- python-2.7.9
